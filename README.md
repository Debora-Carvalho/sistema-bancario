## Para executar localmente

Clone o projeto

```bash
  git clone https://github.com/Debora-Carvalho/sistema-bancario.git
```

Entre no diretório do projeto

```bash
  cd projeto-base
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
